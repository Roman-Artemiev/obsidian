# LinkedIn Post Rewrite

## Overview

Use this skill on existing drafts.

Preserve the idea. Improve the execution.

## Rewrite Priorities

1. Fix the opening
2. Clarify the main point
3. Remove redundant lines
4. Repair paragraph flow
5. Add specificity where the claims are weak
6. Tighten the CTA

## Workflow

1. Identify the post goal and audience.
2. Extract the single core message.
3. Cut filler, warm-up, repetition, and vague abstractions.
4. Rebuild the structure using a simple flow:
   hook, context, insight, implication, CTA.
5. Replace broad claims with examples, mechanisms, or observations.
6. Return a cleaner draft plus short notes on what changed.

## Rewrite Rules

- Keep the strongest idea and remove side quests.
- Do not preserve bad formatting for the sake of fidelity.
- Combine one-line paragraphs when they create noise instead of emphasis.
- If the post sounds “smart” but not useful, rewrite toward clarity.
- If the CTA is generic, make it concrete or cut it.

## Output Pattern

- `Rewritten draft`
- `What changed`
- optional `Alternative hook options` when the opening is still flexible

## Common Rewrite Cases

- too many line breaks
- one huge paragraph
- generic “thought leadership” tone
- too much setup before the point
- too many ideas competing in one post
- weak closing ask
