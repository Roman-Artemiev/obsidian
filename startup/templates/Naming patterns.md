### Naming Directions

2. Invented / brandable
3. Compound
4. Metaphor / symbolic
5. Technical / modern

### Name Generation Prompt

```text
You are a senior brand strategist. Based on the startup brief below, generate naming options for a product, app, or service.

Requirements:
- Create names that are easy to pronounce
- Create names that are easy to remember
- Favor names that feel credible and brandable
- Avoid awkward spelling and confusing pronunciation
- Generate multiple styles: descriptive, invented, compound, metaphorical, and modern
- For each name, include a short rationale
- Prioritize names likely to have available domains, especially .com, then .cz, .app, and .dev

Startup brief:
- Product / app / service: [fill in]
- One-line description: [fill in]
- Target customer: [fill in]
- Main problem solved: [fill in]
- Category: [fill in]
- Brand personality: [fill in]
- Country / market focus: [fill in]
- Constraints: [fill in]

Return:
1. 20-30 name ideas grouped by naming pattern
2. A shortlist of the best 5-10 options
3. A short note on pronunciation and memorability for each shortlisted option
4. Suggested domain checks for each shortlisted option
```
